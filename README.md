# Jeffrey’s Git Page

This is a personal site built with [Quarto](https://quarto.org/) and published to GitHub Pages (`jeffhding.github.io`).

## Local preview

From this directory:

```bash
quarto preview
```

To build static output (written to `docs/` per `_quarto.yml`):

```bash
quarto render
```

Requires [Quarto](https://quarto.org/docs/get-started/) installed locally.
